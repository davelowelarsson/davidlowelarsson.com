fredagslistan started as a Slack ritual: every Friday, a theme appears, people drop songs, and the playlist grows.

The weak link was boring and human: many songs never made it from Slack into Spotify. So the playlist was always a little wrong.

I wrote a small Python cronjob to do the transfer for us. It has been running for years now. Quietly. No heroics. Exactly what I wanted.

## One run

### 1. Fetch from Slack

The job reads the Friday channel, extracts track links, and normalizes IDs. This is mostly input hygiene, because chat data is messy in ways APIs are not.

<ProcessStepCard step={syncSteps.fetchSlack} />

### 2. Compute the diff

Next step is simple set math: what exists in Slack but not yet in Spotify. No scoring model, no ranking, no "smart" guesswork. Just the gap.

<ProcessStepCard step={syncSteps.diff} />

### 3. Add missing songs

Missing tracks are appended to the playlist. If a song is already there, the run leaves it alone. Re-runs are intentionally boring.

<ProcessStepCard step={syncSteps.addSpotify} />
