This is a test fixture, not a real post — it exists so a human can eyeball
real playback on the preview deployment (issue #10). It stays `draft: true`
forever, so the production build (which excludes drafts) never ships it. No
automated test loads the real R2 file — Container tests cover `<Video />`'s
markup instead, and this fixture's own `<video>` attributes are what a
human/Playwright would check without waiting on the network.
